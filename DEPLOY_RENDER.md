# Бесплатный запуск бота без твоего компьютера

Самый простой вариант: Render Free Web Service + Telegram webhook. Компьютер после деплоя можно выключать.

Важно: бесплатные web services могут засыпать при простое. Первое сообщение после паузы иногда обрабатывается медленнее, но бот проснется от webhook-запроса Telegram.

## Что понадобится

1. Аккаунт GitHub.
2. Аккаунт Render: https://render.com
3. Telegram bot token.
4. OpenAI API key.

## Шаг 1. Залей проект на GitHub

1. Открой https://github.com
2. Создай новый репозиторий, например `ai-sales-qa-agent`.
3. Загрузи туда все файлы этого проекта.

Не загружай файл `.env`. Он уже добавлен в `.gitignore`.

## Шаг 2. Создай сервис на Render

1. Открой Render Dashboard.
2. Нажми `New`.
3. Выбери `Blueprint`.
4. Подключи GitHub-репозиторий с проектом.
5. Render найдет файл `render.yaml`.
6. Нажми `Apply`.

Render создаст web service на бесплатном тарифе.

## Шаг 3. Добавь секреты

В Render открой сервис `ai-sales-qa-agent` -> `Environment`.

Добавь значения:

```env
TELEGRAM_BOT_TOKEN=твой_токен_бота
OPENAI_API_KEY=твой_openai_ключ
ADMIN_USER_IDS=твой_telegram_id
```

`WEBHOOK_SECRET` Render сгенерирует сам.

## Шаг 4. Запусти Deploy

Нажми:

```text
Manual Deploy -> Deploy latest commit
```

Дождись статуса `Live`.

## Шаг 5. Проверь Telegram

Открой:

```text
https://t.me/Elevate_salles_bot
```

Напиши:

```text
/start
```

Если видишь приветствие с командами `/new`, `/done`, `/stats`, значит бот работает из облака.

## Как пользоваться

1. Менеджер отправляет скриншоты.
2. Если это альбом, бот сам сгруппирует изображения.
3. Если это отдельные картинки, после всех скринов нужно отправить `/done`.
4. Бот пришлет оценку, ошибки и рекомендации.

## Про скриншоты

Скриншоты удаляются после анализа:

```env
DELETE_IMAGES_AFTER_ANALYSIS=true
```

В базе остается только текст оценки, score, ошибки и рекомендации.

## Про бесплатность

Render Free подходит для теста и небольшого использования. Если бот станет активно использоваться каждый день многими людьми, бесплатный тариф может начать упираться в лимиты.

## Если нужна постоянная база

SQLite на бесплатном облаке может сбрасываться при пересборке сервиса. Для постоянного leaderboard лучше создать бесплатную Supabase Postgres базу и заменить `DATABASE_URL` на:

```env
DATABASE_URL=postgresql+asyncpg://USER:PASSWORD@HOST:PORT/postgres
```
